# Smart Assistant
**Smart Assistant** - это Flask-приложение для управления задачами, которое позволяет пользователю:
- регистрироваться и входить в аккаунт
- создавать, редактировать и удалять задачи
- распределять задачи по категориям
- отслеживать статус задач, дедлайн
- просматривать аналитику продуктивности
- взаимодействие с приложением через навык Яндекс Алисы

Проект разработан с использованием **Flask, SQLAlchemy, HTML-шаблонов** и, конечно, **логики Python**.

## Основной функционал
- Регистрация и авторизация пользователей
- Личный кабинет пользователя
- Разделение задач на категории
- Умная логика приоритетов задач
- Аналитика выполнения задач
- Интеграция с навыком Яндекс Алисы
- Подготовка к хостингу

## Структура проекта
- `app/` — основная логика приложения  
- `app/auth/` — регистрация, вход, выход  
- `app/main/` — главная страница и dashboard  
- `app/tasks/` — задачи 
- `app/categories/` — категории  
- `app/profile/` — профиль пользователя и код привязки Алисы  
- `app/analytics/` — статистика и аналитика  
- `app/alice/` — webhook для навыка Алисы  
- `app/models/` — модели БД (`User`, `Task`, `Category`, логи)  
- `app/services/` — бизнес-логика (задачи, аналитика, Алиса)  
- `app/templates/` — HTML-шаблоны  
- `app/static/` — CSS  
- `migrations/` — миграции БД  
- `run.py` — запуск приложения  
- `requirements.txt` — зависимости  
- `Procfile` — запуск (Render???? потом посмотрим)
- `README.md` — документация  

## Подробнее про Яндекс Алису
-По сути Алиса умеет:
1) Создавать задачу голосом, добавляя ее в БД
2) Показывать задачи
3) Завершать задачу
4) Показывать ближайшую / важную задачу

## Запуск
- pip install -r requirements.txt
- flask db upgrade
- python run.py
